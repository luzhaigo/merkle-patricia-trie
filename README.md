# Merkle Patricia Trie

A Go implementation of the Merkle Patricia Trie, a data structure used in Ethereum to efficiently store and verify key-value pairs.

This repository serves as a learning and practice project rather than a production-ready implementation.

## Reference

This project is based on the following reference implementation:

- [merkle-patricia-trie](https://github.com/zhangchiqing/merkle-patricia-trie) by zhangchiqing

## Getting Started

### Prerequisites

- [Go](https://go.dev/dl/) 1.25.5 or later

### Installation

```bash
git clone https://github.com/<your-username>/merkle-patricia-trie.git
cd merkle-patricia-trie
go mod tidy
```

### Running Tests

```bash
go test ./...
```
